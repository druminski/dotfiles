# My dotfiles

## Install

```
git clone https://github.com/druminski/dotfiles.git ~/Projects/dotfiles

ln -sv ~/Projects/dotfiles/.sshrc ~
ln -sv ~/Projects/dotfiles/.sshrc.d ~

ln -sv ~/Projects/dotfiles/.gitconfig ~
ln -sv ~/Projects/dotfiles/.gitignore ~

ln -sv ~/Projects/dotfiles/.config/fish/config.fish ~/.config/fish/config.fish
ln -sv ~/Projects/dotfiles/.config/fish/fish_variables ~/.config/fish/fish_variables
ln -sv ~/Projects/dotfiles/.config/fish/functions/ ~/.config/fish/functions/ports.fish
ln -sv ~/Projects/dotfiles/.config/fish/functions/ ~/.config/fish/functions/genpass.fish


```
